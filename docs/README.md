# Dogs of Make School API

## How to Run This Code

Clone this repository and run:

```
npm start
```

# Dogs Endpoints

## Get Homepage

Send a request to `/` to get the following response:

```json
{
```

## Create a New Dog

Send a POST request to `/dogs/` with the following information:

```
name: string,
breed: string,
color: string,
age: number,
owner: string
```
